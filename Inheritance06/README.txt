Challenge
=========

1) You cannot change anything the Program class.

2) You must add the classes and their implementations that will 
   produce the output in the output.png file.

3) You will use the Random class to generate an amount of 
   damage generated by the given character's Attack() method.
   It will use the character's Strength property as the upper
   boundary of the attack amount.

4) The Warrior's attack will also include the Bonus, which
   will be displayed to the end user.  The text displayed to
   screen will be different from the Player.  Refer to the
   output.png file in this project.

5) The Wizard's attack will deplete a random amount of 
   Energy between 1 and 10.  It will display two lines of
   output.  Refer to the output.png file in this project.
